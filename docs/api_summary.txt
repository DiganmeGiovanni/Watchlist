GET /login -> Retrieve temporary access token and user details
GET /lists
    Retrieve all the lists of a given
    user (Identified by its access token)
GET /lists/{id}/movies
    Retrieve the specified list and its contents
    (Image, title, genres and release year)
POST /lists{id}/movies
    Adds a movie to specified list
GET /lists/{id}/movies/{id}
    Retrieves all the details of given
    movie
POST /lists/{id}/movies/{id}/share
    Shares specified playlist
